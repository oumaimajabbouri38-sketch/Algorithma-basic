# 📘 Algorithm Basics Collection
![Language](https://img.shields.io/badge/language-C-blue) ![Level](https://img.shields.io/badge/level-beginner-green) ![Status](https://img.shields.io/badge/status-learning-yellow) ![Algorithms](https://img.shields.io/badge/topic-algorithms-orange)

## 📌 Description
This repository contains a set of basic algorithm exercises written in pseudocode style.  
It is designed to help beginners understand core programming concepts such as input/output, conditions, loops, and arrays.

---

## 📂 Projects Included

### 🔹 Basic Input / Output
- **NameDisplay** → Read and display a name

---

### 🔹 Arithmetic Operations
- **SumTwoNumbers** → Add two integers  
- **AverageCalculator** → Calculate the average of 3 numbers  
- **SquareNumber** → Compute the square of a number  
- **SquareRootCalculator** → Compute the square root of a number  
- **VatCalculator** → Calculate total price with VAT  

---

### 🔹 Conditional Statements
- **EvenOrOddChecker** → Check if a number is even or odd  
- **NumberSignChecker** → Check if a number is positive, negative, or zero  
- **PassFailChecker** → Check if a student is admitted or not  
- **MaximumOfThreeNumbers** → Find the largest number among three  
- **MinimumOfThreeNumbers** → Find the smallest number among three  

---

### 🔹 Loops
- **SumFrom1ToN** → Calculate the sum from 1 to n  

---

### 🔹 Arrays & Searching
- **LinearSearchInArray** → Search for an element in an array (linear search)

---

## 🎯 Learning Objectives
- Understand algorithmic thinking  
- Practice pseudocode writing  
- Learn control structures (conditions & loops)  
- Work with basic arrays and searching techniques  

---

## 🧠 Key Concepts Covered
- Input / Output  
- Variables & Data Types  
- Conditions (if / else)  
- Loops (for)  
- Arrays  
- Basic Mathematical Operations
